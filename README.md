# ProFix Handyman Website

A modern, responsive website for a Handyman business.

## Project Structure
- `index.html`: Main HTML file
- `style.css`: Styles and animations
- `script.js`: Interactivity and form handling

## How to Deploy

### Option 1: Netlify Drop (Easiest)
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop).
2. Drag and drop the `handyman_site` folder into the browser window.
3. Netlify will instantly deploy your site and give you a live URL (e.g., `profix-handyman.netlify.app`).
4. You can later connect a custom domain (e.g., `yourname.com`) in the site settings.

### Option 2: GitHub Pages (Best for updates)
1. Initialize a Git repository:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```
2. Create a new repository on GitHub.
3. Push your code to GitHub.
4. Go to Repository Settings > Pages and select the `main` branch as the source.

### Option 3: Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the terminal and follow the prompts.
